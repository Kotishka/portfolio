How a Computer Works

Fall 2019 - Spring 2020
Soldering, Logic Gates, Circuits

How a Computer Works is a set of displays, targeted for K-12 students, covering the topics of basic logic gates, transistors, and binary numbers. This project was created under the Mobile STEM lab program for Vertically Integrated Projects (VIP).

The basic logic gates display covers NOT, OR, NOR, AND, and NAND gates. The students can play with the different gates to figure out which one is which with guided learning materials.

The binary numbers display allows students to interact with three binary based buttons that translate to a decimal number, up to 7.
